Project Summary

A bespoke, single-page brand website for Marrow Butchery — a premium butcher and charcuterie house. The site features a hero image slider, slim left-side navigation drawer, brand story, a filterable product selection grid (15 cuts across Beef / Pork / Lamb / Poultry / Seafood / Charcuterie), newsletter capture and a multi-column footer. Hand-coded in HTML, CSS and vanilla JavaScript with full responsive support and image-failure fallback handling.
