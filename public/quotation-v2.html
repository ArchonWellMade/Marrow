Project Summary

Phase 2 expands the original single-page brand site (delivered under Q-2026-002) into a full multi-page experience and migrates the codebase from hand-written HTML into a production Next.js application deployed on Vercel.

Five new editorial pages were written, designed and built (Our Craft, Selection, Dry-Aging, Wholesale, Visit Us). A complete product-detail system was added covering twelve hand-selected cuts across Beef, Pork, Lamb, Poultry and Charcuterie, each with its own image gallery, specifications, copy and related items rail. A comprehensive responsive overhaul brings full support for every modern device — iMac, MacBook, iPad portrait & landscape, iPhone, iPhone SE, Android phones — with iOS safe-area awareness and touch-device optimisations.

Delivered & live on Vercel. Source code is published to GitHub at github.com/ArchonWellMade/Marrow and auto-deploys on every push to main. Twenty-four routes are statically pre-rendered at build time; first-load JS is ~102 kB on every page.
